## Exercises

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/cicwi/denoising-tutorial/blob/main/Exercises.ipynb)

Open `Exercises.ipynb` in Google Colab or run it locally using:

```
# Create a virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows, use `.venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook Exercises.ipynb
```
